# Laptop setup

## Update osx

1. Update mac software:

        sudo softwareupdate -i -a --restart

2. Install Apple's command line tools:

        xcode-select --install

3. Reboot, check again.

## Installation

1. Clone this repo
2. Run `sh setup`
3. Reboot

## Config

1. Clone dotfiles
2. Run dotfiles setup
3. Reboot
